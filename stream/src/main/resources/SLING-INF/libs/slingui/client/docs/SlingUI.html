(static) createResource(base, path) → {Q.promise}

Returns a Resource object for data located at the given path. This specification does not define the location for resources or the semantics for resource paths. For an implementation reading content from a Java Content Repository, the path could be a javax.jcr.Item path from which the resource object is loaded.
Parameters:
Name Type Description
base Resource The base Resource against which a relative path argument given by path is resolved. This parameter may be null if the path is known to be absolute.
path string The path to the resource object to be loaded. If the path is relative, i.e. does not start with a slash (/), the resource relative to the given base resource is returned. The path may contain relative path specifiers like . (current location) and .. (parent location), which are resolved by this method.
